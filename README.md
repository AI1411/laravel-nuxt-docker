### Installing nuxt:
1. Remove old client directory:
```
sudo rm -rf client
```

2. Create fresh nuxt project
```
yarn create nuxt-app client
```


### Installing fresh laravel:
1. Remove old client directory:
```
sudo rm -rf api
```
2. Create fresh laravel project
```
yarn create nuxt-app client
```
3. Set up permissions
